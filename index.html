Toric Calabi-Yau Database

 New: Orientifold CY3!

We have added a large class of Calabi-Yau orientifolds, their orientifold planes, and even-/odd-parity Hodge numbers. This work will be coming soon to the arXiv, please cite us!


Search functionality for orientifolds is still in progress, but the data can now be downloaded in bulk from here. Files with the suffix .invol.json contain all orientifold data.

This database is based on arXiv:1411.1418. Please cite us.
